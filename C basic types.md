### Integers

| type        | bytes | scanf       | printf    |
| ----------- | ----- | ----------- | --------- |
| `int`       | 4     | `%d` `%i`   | `%d`      |
| `char`      | 1     | `%c` `%hhd` | `%c` `%d` |
| `short`     | 2     | `%hd`       | `%d`      |
| `long`      | 4/8   | `%ld`       | `%ld`     |
| `long long` | 8     | `%lld`      | `%lld`    |
### Float

| type          | bytes | scanf | printf       |
| ------------- | ----- | ----- | ------------ |
| `float`       | 4     | `%f`  | `%f`         |
| `double`      | 8     | `%lf` | `%lf` `%f`   |
| `long double` | 10    | `%Lf` | `%llf` `%Lf` |
| all           |       |       | `%g` `%e`    |
