#programming 
### Structure
#### Include
- Inserts header file text
- Useful to [[C include|distribute large program]]
#### comments
`/* */` - multistring  
`//` - single string
### Example
```c
#include <stdio.h>
int x; // global variables here

int main() 
{
	int a; // local func variable
	printf("Hello world!\n"); // print f - format
	printf("I like %d %d\n", x, y) // %d for int
	return 0;
}
```
