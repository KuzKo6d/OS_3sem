В современных Linux дистрибутивах существует файл `/proc/sys/kernel/yama/ptrace_scope`. В этом файле хранится настройка безопасности для [[2.4 Реализация процессов в ОС UNIX#Ptrace - Взаимодействие "Главный - подчинённый"|ptrace()]].


| Value | Meaning                                                                                            |
| ----- | -------------------------------------------------------------------------------------------------- |
| 0     | Можно трассировать любой процесс, работающий под тем же UID (по умолчанию во многих дистрибутивах) |
| 1     | Ограниченная трассировка. Можно трассировать только прямых потомков.                               |
| 2     | Только администратор. Трассировка разрешена только процессам с мандатом `CAP_SYS_PTRACE`           |
| 3     | Полностью отключён. Никто не может использовать ptrace.                                            |
