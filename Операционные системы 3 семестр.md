# Котята и умные мысли или как их там
## Заметки с первой лекции
> Unix - одно из лучших изобретений человека за последний век.
### Чем хорош Unix?
- Мобильность/кроссплатформенность
- Открытый исходный код
- Одна из первых систем на базе языка высокого уровня (машинно независимый)
- Потребляет минимум ресурса
### С как машинно-независимый ассемблер
- Нестрогая [[Типизации языков программирования|типизация]]
- Прямой доступ в память
### Изучение C в рамках Unix
- Будем осваивать *многопроцессность*
- Будем использовать *системные вызовы*
